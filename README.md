Spring boot & Spring security 활용한 게시판

개발 환경: eclipse 
개발 스킬: java11
프레임워크: Spring boot 2.7.17
데이터베이스: MySQL
java-db mapping: JPA

1) Spring boot Board게시판
   - CRUD 기능
   - 회원가입 및 로그인
   - 게시판 목록 페이징 처리
  
2) Spring security 활용
   - security활용하여 접근 보호
   - 비밀번호 암호화 및 복호화 기능 구현
  
3) 카카오 로그인 OAuth2.0 기능 구현
